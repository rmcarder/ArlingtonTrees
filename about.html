About

Data

This site uses data from two sources:

  • Arlington County's open data portal. Tree Canopy data is available here and building footprints are available here.
  • Population, race, and incoome date are from the US Census Bureau's American Community Surevy (2014-2018 5 year estimates).
  • The basemap was built by Mapbox using data from OpenStreetMap.

Methodology
  • ACS data at a block group level was obtained using the R package tidycensus. Block group geometry was also downloaded in this process. Tree Canopy and building footprint data shapefiles were manually downloaded from Arlignton County's open GIS portal.
  • Areas of tree canopy and building footprints withing each census block were calculated using the R package simple features. The area calculations were joined back to the census block geometry and together exported to a topojson file using the R package geojsonio.
  • The three polygon layers (census blocks with associated data, tree canopy, and building footprints) were uploaded to Mapbox Studio, then displayed and styled for the web using Mapbox GL.
